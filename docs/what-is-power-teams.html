Built for autonomous development

Set the direction once, then let specialised agents plan, build, review, and iterate with shared state.

🧠

Manager Agent

Breaks goals into actionable tasks, chooses priorities, and coordinates the loop.

🔨

Worker Agent

Implements focused changes without becoming its own reviewer or product owner.

🛡️

Reviewer Agent

Checks quality, catches regressions, and verifies fixes before the next iteration.

📊

Live Dashboard

React + Vite UI for monitoring agent activity, progress, and runtime state.

Architecture

Task Hounds is backend-agnostic: OpenCode, Hermes, OpenClaw, or your own backend through a clean interface.

Why Task Hounds?

Instead of asking one agent to do everything, Task Hounds separates planning, execution, and review.

Quick Start

Install the Python package, build the optional dashboard, then initialise the database.

Install

Clone the project and install the backend package in editable mode.

Build Dashboard

Install frontend dependencies and build the React + Vite dashboard.

Run

Start one cycle, continuous mode, or auto-release mode.

# Clone the repository
git clone https://github.com/catowabisabi/power-teams.git
cd power-teams

# Python backend
pip install -e .

# Optional dashboard
cd frontend
npm install
npm run build
npm run dev

# Database
python -m power_teams db init

# Run a task
python -m powerteams run \
  --task "Implement user authentication" \
  --mode continuous \
  --max-iterations 50

Roadmap
